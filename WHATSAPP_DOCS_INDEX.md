## 🚀 Getting Started Checklist

- [ ] Read WHATSAPP_QUICK_REFERENCE.md (5 min)
- [ ] Get WhatsApp credentials from Meta (see WHATSAPP_SETUP.md)
- [ ] Add environment variables to .env.development
- [ ] Restart server: `npm run dev:server`
- [ ] Test health endpoint: `curl http://localhost:3000/api/whatsapp/health`
- [ ] Send test message: `curl -X POST http://localhost:3000/api/whatsapp/test ...`
- [ ] Test full flow in app (submit reservation)
- [ ] Deploy to Vercel (add env vars to Project Settings)
- [ ] Test on production URL
- [ ] Monitor WhatsApp notifications
